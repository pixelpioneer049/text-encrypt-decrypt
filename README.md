# Text Encrypt/Decrypt App

1. Install project
Download git repository on the local machine and install node packages

```
npm install
```

2. Run project

```
npm start
```

3. Deploy project
Build the project.
```
npm build
```
Deploy the project using `surge`
```
surge
```

4. Published one can be found here.
http://encrypt.vanguard-shell.surge.sh/
